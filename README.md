## 📖 Описание проекта

**TaskBoard** — это интерактивная доска задач с полным CRUD-циклом:

- ✅ Просмотр списка задач (сортировка: невыполненные сверху)
- ✅ Добавление новых задач (название + описание)
- ✅ Отметка о выполнении (через чекбокс)
- ✅ Редактирование задачи (встроенный режим)
- ✅ Удаление задачи (с подтверждением)
- ✅ Фильтрация: Все / В работе / Выполненные
- ✅ Счётчик выполненных задач
